# 梵响后台API-梵响公寓管理系统

## 目录结构
* application 代码目录
* application/vendor composer引入文件目录
* application/composer.json composer文件
* public web根目录
* db 数据库脚本备份目录
* system 框架文件目录【勿动】

## 基本说明
* 前后端分离
* LNMP = CentOS + Tengine + MySQL + PHP7
* CodeIgniter + Eloquent ORM + PHP7
* MySQL5.6
* Redis
* Beanstalkd
* Tengine + PHP-fpm
* ECS + OSS + CDN + SLB
* 此项目为前端提供API，使用jwt作为登录验证方式
